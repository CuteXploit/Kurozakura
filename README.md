## 🎭 About Kurozakura

**Kurozakura** (黒桜 - *Black Cherry Blossom*) is a minimalistic yet powerful PHP web shell designed for **legitimate server administration and security testing**.

> ⚠️ **Disclaimer**: This tool is for **authorized testing and educational purposes only**.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🖥️ **Command Execution** | Run system commands directly from web interface |
| 📁 **File Upload** | Upload files to any directory with write permissions |
| 📍 **Directory Navigation** | Browse and work in any writable directory |
| 🔍 **System Info** | Displays OS, Server IP, Wget & Curl availability |
| 🎨 **Clean UI** | Simple, functional interface with no bloat |
